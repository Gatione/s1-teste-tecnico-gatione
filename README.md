# Documentação

## Clonar o projeto

### Utilize o comando no terminal `git clone (SSH Key)` deste repositório para criar uma cópia deste projeto em uma pasta local

## Instalar dependências

### Dentro da pasta clonada digite o comando `yarn` ou `yarn install` para instalar todas as dependências do projeto

## Rodar o projeto localmente

### Utilize o comando no terminal `yarn start` para poder rodar a aplicação em seu navegador

### Preencha os campos no navegador como indicado e aperte no botão Simular
